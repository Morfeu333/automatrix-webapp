### Variáveis Dinâmicas:
- `[BUSINESS_SIZE]`: Individual/Small/Medium/Large Enterprise
- `[TECHNICAL_LEVEL]`: Beginner/Intermediate/Advanced/Expert  
- `[PRIMARY_GOAL]`: Save time/Improve service/Scale marketing/etc.
- `[BUDGET_RANGE]`: Under $500/$500-1500/$1500-5000/etc.
- `[RECOMMENDED_OPTION]`: Meeting/Shop/Community

### Lógica de Recomendação:
**Meeting Recommended If:**
- Budget > $5,000 AND Business Size = Large Enterprise
- Implementation Preference = "Full Service"

**Shop Recommended If:**
- Budget $500-5,000 AND Technical Level = Intermediate/Advanced
- Primary Goal = Scale/Automate specific tasks

**Community Recommended If:**
- Budget < $1,500 AND Technical Level = Beginner
- Implementation Preference = "Learn myself"
